# Project EAST Backend

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Run in development mode:
   ```sh
   npm run dev
   ```
3. Build for production:
   ```sh
   npm run build
   ```
4. Start the built server:
   ```sh
   npm start
   ```
5. Run backend tests:
   ```sh
   npm test
   ```

The backend runs on port 4000 by default.
